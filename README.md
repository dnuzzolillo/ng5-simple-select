# ng5-simple-select
Angular 5 simple select multiple component

 Instalation:
 Just import on app.module.ts
 - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
 How to use:
 <app-select [options]="options_array" (onSelect)="onSelectFunction($event)" (onDelete)="onDeleteFunction($event)"></app-select>
